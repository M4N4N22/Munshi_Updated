### Railway-specific considerations

| Topic | Assessment |
|-------|------------|
| Build time | **LONG** — `torch` + `sentence-transformers` (~2–5 min) |
| Memory | **HIGH** — may need Railway plan with ≥2GB RAM |
| `PORT` binding | Use `$PORT` in start command (Railway injects) |
| `PYTHONPATH` | Working dir = `ml/` — imports resolve without extra config |
| Private networking | Backend `ML_URL` → Railway private domain |

### Blockers

| Issue | Severity | Mitigation |
|-------|----------|------------|
| Large ML deps | MEDIUM | Upgrade Railway plan; consider slim requirements later |
| No `PORT` in code | LOW | Start command uses `$PORT` |

**Verdict:** Railway ML source build — **READY**
